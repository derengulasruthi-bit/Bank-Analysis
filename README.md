📊 Loan Data Analysis Dashboard

📌 Project Overview

This project focuses on analyzing loan data to understand customer behavior, loan distribution, and repayment trends.

🎯 Objectives

- Analyze total loan applications
- Track funded amount vs recovered amount
- Identify year-wise growth trends
- Understand loan status (Fully Paid, Current, Charged Off)

🛠 Tools Used

- Excel (Data Cleaning)
- SQL (Data Extraction & Queries)
- Power BI / Tableau (Dashboard & Visualization)

📊 Key KPIs

- Total Applications
- Total Funded Amount
- Total Amount Received
- Remaining Amount to be Recovered

📈 Insights

- Identified increase/decrease in loan trends over years
- Analyzed customer segments based on home ownership
- Tracked repayment efficiency using loan status
- Loan Distribution Trend
- Loan Status Analysis
- Risk Insights
